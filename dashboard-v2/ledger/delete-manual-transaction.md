# How to Delete a Manual Transaction

Remove manually added transactions from your Tres Finance ledger when they're no longer needed.

---

## Before You Start

- You need admin or write permissions to delete transactions
- Only manual transactions that haven't been deleted already can be removed
- This action is irreversible, but deleted transactions can be restored if needed

## Steps

### Step 1: Navigate to Your Ledger

Go to the Ledger section of your Tres Finance dashboard.

![Navigate to Ledger](screenshots/step_01.png)

### Step 2: Locate the Transaction to Delete

Browse or search through your transaction list to find the manual transaction you want to remove.

![Find Transaction](screenshots/step_02.png)

**Tip**: Use the search and filter options to quickly locate specific transactions.

### Step 3: Access the Transaction Menu

Click on the actions dropdown menu for the transaction you want to delete, then select "Delete Transaction."

![Select Delete Option](screenshots/step_03.png)

**Note**: The delete option only appears if you have the necessary permissions and the transaction hasn't been deleted previously.

### Step 4: Confirm the Deletion

A confirmation dialog will appear asking you to verify the deletion. Click "Confirm" to proceed with removing the transaction.

![Confirm Deletion](screenshots/step_04.png)

**Important**: This action is irreversible, so make sure you want to permanently remove this transaction.

### Step 5: Verify the Transaction is Removed

The transaction will be removed from your ledger view, and you'll see a success message confirming the deletion.

![Transaction Deleted](screenshots/step_05.png)

**Note**: If you need to recover a deleted transaction, look for the "Restore transaction" option in the actions menu for deleted items.

---

**Need Help?** Contact Tres Finance support if you encounter any issues or need to restore a deleted transaction.
