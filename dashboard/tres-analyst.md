# Tres Analyst - AI-Powered Financial Assistant

## Overview

**Tres Analyst** is TRES's intelligent AI-powered financial assistant that enables you to interact with your financial data through natural language conversations. Instead of navigating complex filters, downloading reports, or manually analyzing data, you can simply ask Tres Analyst questions and get instant, accurate answers about your portfolio, transactions, assets, and reports.

Tres Analyst is designed to:

- **Understand Financial Nuance**: Comprehends complex financial terminology and transactions specific to digital assets
- **Deliver Pinpoint Accuracy**: Provides precise, data-driven answers based on your actual financial records
- **Save You Time**: Eliminates the need to download reports, apply filters, and manually analyze data

## Where You Can Use Tres Analyst

Tres Analyst is accessible throughout the TRES platform in several key locations:

### 1. Global Access (Bottom-Right Corner)

**Location:** Fixed button in the bottom-right corner of every page (Overview, Ledger, Accounts, Assets, Reports, etc.)

**Features:**

- **Always Available**: Accessible from any page in the application
- **Contextual Intelligence**: Automatically understands what page you're on and provides relevant prompts
- **Minified Chat**: Compact view that can be expanded to full chat interface
- **Persistent Conversations**: Maintains chat history across sessions

**How to Access:**

1. Look for the blue circle icon with the "Tres Analyst" icon in the bottom-right corner
2. Click the button to open the chat interface
3. The chat will load with context-aware prompts based on your current page

## How Tres Analyst Works

### Starting a Conversation

**Method 1: Using Suggested Prompts**

1. **Click the Tres Analyst button** in any location (global button, report row, transaction, etc.)
2. **Browse suggested prompts** organized by category:
   - **Networth Analysis**: Daily networth, account-level networth, asset class breakdowns
   - **Exposure Analysis**: Asset-specific exposure over time
   - **Transaction Queries**: Platform-specific, contact-specific, and date-range filtered transactions
   - **Data Operations**: Tagging, wallet creation, data collection triggers
3. **Select a prompt** that matches your question
4. **The prompt will be filled in** with placeholders for dates, addresses, or other variables
5. **Fill in the variables** (e.g., enter dates, select asset types, choose wallets)
6. **Submit your query** to get instant results

**Method 2: Asking Your Own Question**

1. **Click the Tres Analyst button**
2. **Type your question directly** in the chat input box
3. **Press Enter** or click Send
4. **Tres Analyst will analyze** your question and provide an answer

### Understanding Tres Analyst Responses

**Response Types:**

1. **Textual Answers**: Plain language explanations and summaries
2. **Data Tables**: Formatted data in tables showing numbers, dates, and amounts
3. **Visualizations**: Charts and graphs when appropriate
4. **Action Suggestions**: Recommendations for follow-up actions
5. **Links**: Clickable links to navigate to relevant pages or transactions

**Response Format:**

- **Answer Header**: Shows what question was asked
- **Data Summary**: Key findings and totals
- **Detailed Breakdown**: Specific data points or transactions
- **Action Buttons**: Quick actions like "View in Ledger" or "Edit Transaction"

### Chat Features

**Message Actions:**

- **Regenerate Response**: Click to get a new answer to the same question
- **Copy Message**: Copy the entire message to clipboard
- **Provide Feedback**: Thumbs up or down to rate the response quality
- **Follow-Up Questions**: Ask additional questions in the same conversation

**Chat History:**

- **View History**: See all previous conversations
- **Resume Conversations**: Pick up where you left off
- **Delete Conversations**: Remove old conversations you no longer need
- **Categorization**: Conversations are organized by type (Suggestions, Transactions, Reports)

**Navigation:**

- **Back to Prompts**: Return to the prompts list to start a new conversation
- **View Context Item**: When chatting about a specific transaction or report, click "View [item]" to see full details
- **Try Another Prompt**: Start fresh with a different question

## Tips for Using Tres Analyst Effectively

### Getting Better Answers

**Be Specific:**

- ✅ Good: "What was my total USDC balance on January 15th?"
- ❌ Vague: "Tell me about USDC"

**Use Proper Dates:**

- ✅ Good: "Show transactions between 2024-01-01 and 2024-01-31"
- ❌ Vague: "Show last month's transactions" (unless you've previously set a default)

**Combine Conditions:**

- ✅ Good: "Show all transactions from Binance over $1,000 in the last 30 days"
- ❌ Too broad: "Show me transactions"

### Understanding Context

**Page Context:**

- Tres Analyst understands what page you're on
- If you're viewing a transaction, it knows which transaction
- If you're viewing a report, it can analyze that report
- Use this context to ask follow-up questions

**Conversation Context:**

- Tres Analyst remembers the current conversation
- You can ask follow-up questions without repeating information
- Reference previous answers in your questions

### Best Practices

**Start with Prompts:**

- The suggested prompts are optimized for the best results
- Use them as templates for your own questions
- Customize variables to match your specific needs

**Review and Refine:**

- Read the answer carefully
- Use the feedback buttons to improve results
- Ask clarifying questions if something is unclear

**Save Important Answers:**

- Copy important information to your notes
- Screenshot insightful responses
- Reference the chat history for previous queries

### Handling Errors

**Common Issues:**

- **"No data found"**: Try broadening your date range or removing filters
- **"Incomplete data"**: Check if all required data is available
- **"Vague question"**: Be more specific about what you want to know

**Getting Help:**

- Use the "Try another prompt" button to restart
- Click "Request prompt" if you need a new prompt type
- Contact support if Tres Analyst consistently fails

## Limitations and Important Notes

### Data Scope

**Available Data:**

- Tres Analyst can access all data in your TRES account
- Includes transactions, balances, assets, wallets, reports
- Can query historical data within your data retention period

**Not Available:**

- External market data or news (unless already integrated)
- Data from accounts you haven't connected
- Future projections or predictions (only historical analysis)

### Accuracy

**AI Limitations:**

- Tres Analyst uses AI to interpret questions
- Sometimes it may misunderstand or require clarification
- Always verify critical information from primary sources
- Use Tres Analyst for insights and queries, not absolute precision

**Data Quality:**

- Answers depend on data quality in your account
- Missing fiat values may affect calculations
- Incomplete transactions may lead to incomplete answers
- Ensure data collection is up-to-date for best results

### Performance

**Query Speed:**

- Simple queries return instantly
- Complex queries may take a few seconds
- Very large date ranges may be slower

**Browsing:**

- You can browse prompts while waiting for responses
- Start new conversations while others finish
- Chat history loads asynchronously

## Security and Privacy

### Data Privacy

**Your Data is Private:**

- All conversations are encrypted
- Data is only accessible within your account
- Tres Analyst doesn't share your information externally
- You can delete conversations at any time

**Access Control:**

- Only users with access to your TRES account can use Tres Analyst
- Permissions follow your account permissions
- Enterprise accounts maintain separate analysis

### Best Practices

**Sensitive Information:**

- Be cautious about asking questions in shared workspaces
- Clear conversation history when needed
- Don't include private keys or passwords in questions

**Audit Trail:**

- Conversation history is maintained
- Useful for compliance and record-keeping
- Can be exported if needed

## Examples of Common Workflows

### Daily Portfolio Check

1. **Open Overview Page**
2. **Click Tres Analyst button** (bottom-right)
3. **Select "What was my total daily networth between [dates]?"**
4. **Enter yesterday's date and today's date**
5. **Get instant networth summary**
6. **Ask follow-up**: "Why did it change?"
7. **Tres Analyst explains** the change with specific transactions

### Investigating a Specific Transaction

1. **Navigate to Ledger Page**
2. **Find a transaction**
3. **Click Tres Analyst icon** on that transaction
4. **Ask**: "Explain this transaction"
5. **Tres Analyst provides** detailed breakdown:
   - Transaction type
   - Assets involved
   - Fees
   - Related transactions
6. **Ask follow-up**: "Show me similar transactions"
7. **Identify patterns** in your transaction history

### Analyzing Monthly Reports

1. **Go to Reports Page**
2. **Generate or open a Balance Trends report**
3. **Click Tres Analyst icon** next to the report
4. **Select**: "What assets had the highest transaction volume?"
5. **Get ranked list** of assets by volume
6. **Ask follow-up**: "What's the trend for the top 3 assets?"
7. **Get visual chart** showing trends over time

### Finding Missing Fiat Values

1. **Open Tres Analyst** (global button)
2. **Ask**: "What transactions are missing fiat values?"
3. **Get list** of transactions needing fiat values
4. **For each transaction, ask**: "Why is this missing a fiat value?"
5. **Tres Analyst explains** the reason (e.g., no price feed, manual entry needed)
6. **Follow the recommendations** to fix the data

### Optimizing Staking Performance

1. **Open Overview → Positions Tab**
2. **Click Tres Analyst**
3. **Ask**: "What are my claimable staking rewards?"
4. **Get list** of rewards ready to claim
5. **Ask**: "What's my average APY across all staking positions?"
6. **Compare performance** across different staking platforms
7. **Get recommendations** for optimization
