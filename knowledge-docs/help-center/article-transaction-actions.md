# Transaction Row Actions

Transaction row actions:

View in explorer - Opens the associated transaction on the relevant blockchain explorer.

Tag transaction - Classify and categorize transactions automatically or manually.

Sync to ERP - Transferring selected transaction data from the ledger to your connected ERP system to ensure financial records are updated and aligned.

Ask TRES Analyst - Some transactions display an analyst icon (an upside-down triangle), which allows you to ask questions and obtain more information about that specific transaction.

Action menu:

Mark as - Favorite, Reviewed, Under review, Not reviewed

Reclassify - Change the transaction's classification or category. For more information on reclassification, refer to the Reclassify guide.

Add note

Copy transaction hash

Copy method ID

Copy transaction link

Delete transaction

Manage sub transactions - View detailed sub-transaction breakdowns, edit, add, remove, or split sub-transactions, and manage linked transactions to ensure accurate and granular transaction accounting.

NOTE: All Transaction actions can be done in bulk using multi-select option.
