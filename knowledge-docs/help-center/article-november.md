# November

### Logos and wire transfer details in invoice configuration

The General Configuration settings now allow users to upload their logo and define standard wire transfer details for automatic inclusion on every invoice generated.

### Favorite Asset classes

Users can now mark specific Asset Classes as favorites. This preference is saved across the platform, allowing for quicker filtering and improved navigation when viewing data.

### New setting called Whitelist Asset Classes

All assets that are NOT on the list will receive 0 fiat value

### Track prices on the platform in BTC and ETH value

Once you set the calculation through the CB configuration, you can then select the currency on the platform.
