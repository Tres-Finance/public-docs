# Integrate BitGo with TRES Finance

Pull wallets, transactions and balances from BitGo into TRES to fetch on-chain data.

The BitGo Access Token works as an API that once imported into TRES will let you:

1. Pull automatically onchain wallets into TRES

2. Track UTXO balances and past transactions.

Important: The term Access Token refers to an API Key.

To obtain an Access Token / API key from BitGo, follow the steps :

1. Login to the BitGo Dashboard (app.bitgo.com)

2. Go to the Account Settings page.

3. Go to the Develop Options section.

4. Choose the Access Token option - Create a long-lived API Access tokens

To create a BitGo Access Token (API Key), follow these steps:

5. Give the API Access Token a memorable name (=TRES Finance API Key).

6. Go to “IP Addresses Allowed”. Here, you need to type TRES’s IP addresses:

44.207.48.224

34.192.2.121

172.190.178.94

172.161.128.154

7. Please also make sure that you assign the correct Permission to the API:

Prime - View Trades

Wallet - View all

8. Read & Agree to the Terms of Conditions

9. Click Add Access Token!

10. Share with TRES the API Key
