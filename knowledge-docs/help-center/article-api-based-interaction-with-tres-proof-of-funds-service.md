# API-Based interaction with TRES Proof of Funds Service

In this article, we will explain step-after-step how to register new addresses and query their balances using PoF APIs.

All the APIs are using MongoDB query filters. The TRES team is available 24/7 to help you combine the relevant query filters to match your requirements.

Our postman contains some simple examples for basic usage.

1. Login into TRES to get an access token

2. Query your current Registered addresses

3. Query balances breakdown (raw data)

4. Query total balances

5. Register new addresses - new address balances' will be collected at the next PoF data collection checkpoint
