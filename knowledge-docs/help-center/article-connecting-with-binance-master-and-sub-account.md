# Connecting with Binance master and sub-account

To enable Tres Finance to monitor the financial activities in your Binance Exchange account, provide the following parameters to the Tres Finance team:

API secret

Related email address

Follow the steps in the following sections to generate these parameters:

### Connect your Binance master account

Follow these steps:

Follow this guide to generate a Binance API key.

When prompted, name your new API key “Tres Finance”.

After you confirm the validity of your new API key, copy and keep a record of both the API key and Secret key. You will need to share them with the Tres Finance team.

Make sure you set API restrictions to read-only.

Under IP access restrictions, select Restrict access to trusted IPs only, delete the 0.0.0.0 address, and add the following IP addresses :​ 45.38.187.157 45.38.187.95 45.38.187.173 45.38.187.236 45.147.70.45 45.147.70.51 45.147.70.71 104.251.94.22 104.251.94.212 212.80.200.19 172.190.178.94 172.161.128.154

Important

Whitelisting an IP address enables network access to relevant IP addresses.

All the above IP addresses must be whitelisted, otherwise, the account connectivity will fail.

### Connect your Binance sub-account

In the Binance web console navigation bar, click Sub Accounts > Account Management.

Click the Account Management tab to see all sub-accounts under your master account. The number of sub-accounts you can own depends on the VIP level of your master account.

Click the API Management tab to create sub-account API keys and API access. The API key should be configured with read-only permissions.
