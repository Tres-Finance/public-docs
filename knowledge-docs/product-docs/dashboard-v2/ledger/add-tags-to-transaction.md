# How to Add Tags to Transactions

Tags help you organize and categorize your transactions for better financial tracking and reporting. You can add existing tags or create new ones directly from the ledger.

## Steps

### Step 1: Navigate to your Ledger
Go to your Ledger page to view all your transactions.

![Step 1](screenshots/step_01.png)

### Step 2: Locate the tag dropdown
Find the tag dropdown button next to the transaction you want to tag. This appears in the tags column of your transaction list.

![Step 2](screenshots/step_02.png)

### Step 3: Search for a tag
Click the dropdown to open it, then use the search box to find an existing tag or type the name of a new tag you'd like to create.

![Step 3](screenshots/step_03.png)

### Step 4: Select or create your tag
- To use an existing tag: Click on it from the search results
- To create a new tag: Type the new tag name and click the "Create" button

![Step 4](screenshots/step_04.png)

### Step 5: Confirm the tag is applied
The tag will immediately appear on your transaction, making it easy to identify and filter later.

![Step 5](screenshots/step_05.png)

## Tips
- You can add multiple tags to a single transaction
- Tags are searchable and help with filtering transactions
- Created tags become available for use on other transactions
