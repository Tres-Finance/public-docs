# How to View Unidentified Addresses

Review blockchain addresses that haven't been assigned to contacts in your Tres Finance dashboard.

## What are Unidentified Addresses?

Unidentified addresses are blockchain addresses that appear in your transaction data but haven't been linked to specific contacts or entities yet. Reviewing these helps you maintain complete transaction records and improve reporting accuracy.

## Steps

### Step 1: Navigate to the Accounts page

Go to the Accounts section of your dashboard.

![Step 1](screenshots/step_01.png)

### Step 2: Click on the 'Unidentified Addresses' tab

Select the "Unidentified Addresses" tab to view addresses that need identification.

![Step 2](screenshots/step_02.png)

### Step 3: Review addresses that need to be identified

You'll see a list of blockchain addresses that haven't been assigned to contacts. From here, you can review each address and decide whether to create new contacts or link them to existing ones.

![Step 3](screenshots/step_03.png)

## Next Steps

Once you've identified addresses you want to assign:
- Create new contacts for unknown counterparties
- Link addresses to existing contacts in your system
- This will improve the accuracy of your transaction categorization and reporting

> **Note**: This feature requires cost basis settings to be enabled for your organization. If you don't see any addresses listed, contact your administrator to ensure this feature is properly configured.
